# ladder_game



#### 사다리타기 게임

|사다리 타기|

재미있는 걸 만들어보자 해서 만들어본 사다리 게임 어플입니다.

사다리는 `paint`를 이용하여 그려주었고, 사다리의 중간 다리는 조금의 확률과 랜덤으로 만들었습니다.

user는 기본 아이콘을 사용하고, `animation`을 이용하여 사다리를 타고 내려가는 움직임을 만들었습니다.

`provider` 라이브러리를 사용하여 관리를 해주었습니다.
